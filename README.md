# 人工智能赋能智慧旅游 - 科普网页项目

这是一个基于 HTML5 和 CSS3 开发的静态科普网页，主题为“人工智能赋能智慧旅游”。项目展示了 AI 在旅游导览、个性化推荐、智能客服和数据分析等场景的应用。

## 项目介绍

- **前端架构**：HTML5 + CSS3 + 原生 JavaScript (无框架，简单易懂)
- **响应式设计**：兼容 PC 端（1200px 居中布局）和移动端（汉堡菜单，单列布局）
- **视觉风格**：科技蓝主色调，卡片式设计，交互友好

## 快速启动 (使用 Docker)

本项目已配置 Docker 环境，可一键启动。

### 前置要求
确保本地已安装 Docker 和 Docker Compose。

### 启动命令

在项目根目录下，运行以下命令：

```bash
docker-compose up -d --build
```

- `up`: 启动服务
- `-d`: 后台运行 (detached mode)
- `--build`: 启动前重新构建镜像

### 访问项目

启动成功后，请在浏览器中访问：

**[http://localhost:3000](http://localhost:3000)**

## 其他启动方式 (直接打开)

如果你不想使用 Docker，也可以直接在浏览器中打开文件。

### 操作步骤
1. 进入 `frontend` 文件夹。
2. 双击 `index.html` 文件，或者右键选择“用浏览器打开”。

### 优缺点分析
- **优点**：
  - **简单快捷**：不需要安装 Docker、Node.js 等任何开发环境，有浏览器就能看。
  - **即改即看**：修改代码后，按 F5 刷新浏览器就能看到效果。
- **缺点**：
  - **地址不规范**：浏览器地址栏显示的是本地文件路径 (如 `file:///D:/...`)，而不是标准的网址 (如 `http://localhost:3000`)。
  - **功能受限**：如果项目中使用了 AJAX 请求或某些需要服务器环境的 API，直接打开可能会报错（**注**：本项目为纯静态页面，不受影响）。
  - **路径问题**：如果代码中使用了绝对路径（如 `/css/style.css`），直接打开可能会找不到文件（**注**：本项目已使用相对路径，无此问题）。

## 停止服务

运行以下命令停止并移除容器：

```bash
docker-compose down
```

## 开发者说明

如果你想手动修改代码：
1. 修改 `frontend` 目录下的 `index.html` 或 `css/style.css`。
2. 刷新浏览器即可看到变化（如果是本地直接打开文件）。
3. 如果使用 Docker 启动，修改后需要重新运行启动命令以重建镜像。
